# Smart Agriculture Monitoring System using ESP32-S3

This project uses ESP32-S3 and Machine Learning for smart agriculture monitoring.

## Features
- Sensor data collection
- Crop prediction model
- ESP32 integration
- TensorFlow Lite deployment
- Data analysis pipeline

## Files Included
- Arduino ESP32 code
- ML training pipeline
- TFLite model
- Deployment simulation
- Experimental datasets
- Performance reports

## Technologies Used
- Python
- ESP32-S3
- TensorFlow Lite
- Arduino IDE
